# 🌞 SunCart – Summer Essentials Store

## 📌 Project Name
SunCart – Summer Essentials Store

---

## 🎯 Purpose
SunCart is a modern summer eCommerce web application where users can browse and purchase seasonal products like sunglasses, summer outfits, skincare items, and beach accessories. Users must log in to access protected features such as product details and profile.

---

## 🚀 Live URL
https://suncart-summer-sale.vercel.app

---

## ✨ Key Features
- Home page with summer banner and featured products
- Product listing page with product cards (image, price, rating)
- Product details page (protected route)
- User authentication (Email/Password + Google Login)
- User registration system
- My Profile page showing user information
- Fully responsive design (mobile, tablet, desktop)
